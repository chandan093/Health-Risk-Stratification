# Health-Risk-Stratification
Machine learning project to predict patient health risk using clinical data, with an interactive Streamlit application.
# 🏥 Health Risk Stratification System

## 📌 Project Overview
This project predicts whether a patient is at **High Risk** or **Low Risk** based on clinical and demographic data.
It uses machine learning techniques to analyze patient information such as age, length of stay, treatment cost, and lab results.
---
## 🎯 Objective
To build an end-to-end data science solution that helps in identifying high-risk patients for better healthcare decision-making.
---
## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
---
## 📂 Dataset
The project uses the following datasets:
- `patients.csv`
- `diagnoses.csv`
- `outcomes.csv`
- `labs.csv`
---
## ⚙️ Features & Workflow
1. Data loading and preprocessing  
2. Merging multiple datasets  
3. Feature engineering:
   - Length of stay calculation  
   - Outcome encoding  
4. Model prediction:
   - Predicts risk level (High / Low)  
   - Outputs probability score  
5. Streamlit web app for user interaction
